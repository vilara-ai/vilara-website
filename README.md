# Vilara.ai Website

Professional website for Vilara - The AI-augmented ERP system that flexibly augments OR replaces traditional ERPs.

## 🎯 Key Features

- **Three Customer Paths**: Augment existing ERP, replace legacy system, or start fresh
- **Interactive Demo**: Fully integrated presentation at /demo.html
- **ROI Calculator**: Real-time savings calculation on homepage
- **Migration Tools**: Complete migration timeline calculator and guides
- **Responsive Design**: Mobile-first, enterprise-ready design
- **Conversion Optimized**: Multiple CTAs based on customer journey

## 📁 Project Structure

```
/
├── index.html                 # Homepage with hero, paths, ROI calculator
├── demo.html                  # Integrated Vilara presentation
├── pricing.html               # Transparent pricing with $0 seat licenses
├── how-it-works.html          # Technical architecture and capabilities
├── migration.html             # Migration guide and timeline calculator  
├── contact.html               # Multi-form contact page
├── solutions/
│   ├── augmentation.html      # ERP augmentation solution
│   ├── replacement.html       # ERP replacement solution
│   └── first-time.html        # First-time ERP solution
└── assets/
    ├── css/styles.css         # Main stylesheet with Vilara branding
    ├── js/main.js            # Interactive features and calculators
    └── images/vilara_logo.png # Vilara logo
```

## 🛠 Deployment

### Vercel/Netlify Deployment
1. Connect GitHub repository
2. Set build command: `npm run build` (if needed)
3. Set publish directory: `/` (root)
4. Deploy to vilara.ai domain

### Form Setup
1. Create Formspree account
2. Replace `YOUR_FORM_ID` in contact.html with actual form IDs
3. Set up different forms for demo, migration, pilot requests

### Analytics Setup
1. Add Google Analytics tracking code
2. Set up conversion goals
3. Monitor user flow through customer paths

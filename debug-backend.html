Backend API Debug Tool

This tool helps debug the backend API endpoints to identify connectivity issues.

1. Test Backend Health/Debug Endpoint

First, let's see if the backend is responding at all.

2. Test Activation API Endpoint

Test if the activation endpoint exists (will fail without token, but should give us info).

3. Test Signup API

Test a real signup to get a token for activation testing.

4. Direct Backend URLs

Try hitting the backend directly (bypassing Vercel proxy).
